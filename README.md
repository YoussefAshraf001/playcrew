<h1 align="center">
  PlayCrew
</h1>

PlayCrew is a fully responsive web application that lets gamers track, share, and explore their gaming journeys. Built with Next.js for fast server-side rendering and smooth navigation, Tailwind CSS for a sleek, responsive interface, Firebase for real-time data storage and synchronization, and NextAuth.js for secure authentication. Users can log their games, track progress, favorite titles, view screenshots, and connect with their crew to compete and level up together. PlayCrew makes gaming social, organized, and fun.
